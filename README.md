# Superstore Sales Dashboard | Power BI

## Project Overview

This project presents an interactive Power BI dashboard built using the Superstore dataset. The dashboard provides insights into sales performance, profit trends, customer segments, regional performance, and product categories to support data-driven business decisions.

## Objectives

* Analyze sales and profit performance
* Identify top-performing regions and categories
* Monitor customer segment contributions
* Track sales trends over time
* Discover business opportunities through data visualization

## Tools Used

* Power BI
* Power Query
* DAX
* Microsoft Excel

## Key Metrics

* Total Sales
* Total Profit
* Total Orders
* Quantity Sold
* Profit Margin

## Dashboard Features

* Sales Analysis
* Profit Analysis
* Region-wise Performance
* Category & Sub-Category Analysis
* Customer Segment Analysis
* Trend Analysis

## Key Insights

* Identified top-performing product categories
* Analyzed regional sales and profit trends
* Compared customer segment contributions
* Evaluated business performance through KPIs
